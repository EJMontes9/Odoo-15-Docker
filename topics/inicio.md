# AAG ERP - Documentación Técnica

Documentación técnica del sistema ERP Odoo 15 para la Autoridad Aeroportuaria de Guayaquil.

## Contenido

### Documentación Técnica de Módulos

Documentación completa de los módulos personalizados del sistema AAG ERP:

- **[Módulo Presupuesto v1.0](modulos/presupuesto/presupuesto-overview.md)** - Gestión de presupuestos gubernamentales
  - Presupuesto inicial y codificado
  - Certificaciones presupuestarias
  - Redistribuciones
  - Cédula presupuestaria (Query Deluxe)
  - Reportes y consultas

### Instalación y Configuración

Guías para la instalación, configuración y gestión del entorno de desarrollo y producción.

- **Instalación y Configuración** - Requisitos, servicios Docker, variables de entorno, estructura de directorios
- **Gestión de Base de Datos** - Métodos de configuración, restauración de backups, regeneración de assets, eliminación segura de registros

### Solución de Problemas

Guías de troubleshooting para problemas comunes del sistema.

- **Generación de PDF** - Configuración de wkhtmltopdf, enlaces simbólicos, modificaciones de código
- **Filestore** - Problemas de estructura de directorios, permisos, scripts de corrección
- **Pantalla en Blanco** - Diagnóstico y solución de pantalla blanca después del login

### Sistemas Contables y Financieros

Descripción funcional de los módulos contables y financieros del sistema.

### Bitácora de Cambios

Registro de todos los cambios, correcciones y mejoras realizadas en los módulos del sistema, organizados por módulo y fecha.

- **Módulo Planificación** - Matriz POA, reformas, techos presupuestarios
- **Módulo Contratos** - Gestión de contratos, migración desde hiring process
- **Módulo Presupuesto** - Mejoras y correcciones al módulo de presupuesto gubernamental
